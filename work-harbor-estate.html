HARBOR ESTATE｜不動産会社ホームページ制作事例

※画像クリックで実際のデモサイトをご覧いただけます

概要

福岡県内の主要エリアに対応する架空の不動産会社「HARBOR ESTATE」のホームページです。
LPではなく、TOP・物件一覧・売買詳細・賃貸詳細・会社概要・お問い合わせなどを含む14ページ構成で制作しました。

想定ターゲット：福岡県内で住まい探しを検討している個人・ファミリー層
目的：物件閲覧、相談導線、会社への信頼形成

制作意図

不動産サイトとして必要な「探しやすさ」と、地域密着型の安心感が伝わる見せ方を意識しました。
ネイビー・白・淡いブルーを基調に、海沿いの暮らしや清潔感が伝わるデザインへ整えています。

設計のポイント

  • 買う・借りるを切り替えられる物件検索画面
  • 売買物件詳細と賃貸物件詳細を分けた情報設計
  • 相談導線を押し売り感なく配置
  • スマホで見やすいカード・フォーム・ギャラリー設計
  • 不動産会社として自然に見える会社概要・法務ページも用意

ページ構成

TOP / 物件一覧 / 売買物件詳細 / 賃貸物件詳細 / 会社概要 / お問い合わせ / お問い合わせ完了 / リノベーション提案 / 民泊・別荘相談 / サンプル物件詳細 / プライバシーポリシー / 特定商取引法に基づく表記 / サイトマップ / 404ページ

使用技術

  • HTML / CSS（セマンティック設計）
  • JavaScript（スライダー、アコーディオン、検索画面、画像ギャラリー）
  • レスポンシブ対応（PC / スマホ）
  • アクセシビリティを意識した画像説明・フォーカス・ボタン設計
