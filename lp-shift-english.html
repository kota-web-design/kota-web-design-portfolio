英語コーチングLP｜90日プランへ導く制作事例

※画像クリックで実際のサイトをご覧いただけます

概要

社会人向け英語コーチングサービスの無料セッション予約を目的としたLPです。
会議・メール・プレゼンなど、仕事で英語を使う場面に合わせて学習内容を整理し、90日プランへつなげる構成にしています。

想定ターゲット：仕事で英語が必要になった社会人、会議・メール・プレゼンに不安があるビジネスパーソン
目的：無料セッション予約・90日学習プラン作成への誘導

制作意図

英語学習サービスのLPでは、「何を学べるか」だけでなく「自分の仕事にどう役立つか」が伝わることが重要です。

ファーストビューでは「会議・メール・プレゼン。仕事英語を、90日で整える。」という具体的な訴求にし、完全オンライン・週1回の個別コーチング・毎日のチャットサポートを早い段階で提示しています。

設計のポイント

本LPは「悩みの共感 → 仕事場面から逆算する学習設計 → 90日ロードマップ → 学習ダッシュボード → 比較表 → コーチ紹介 → モデルケース → 料金 → 無料セッション → FAQ」の流れで構成しています。

学習内容を抽象的に見せるのではなく、仕事場面別の課題、優先スキル、週ごとのロードマップを見せることで、受講後のイメージを具体化しています。

想定される効果

「英語を勉強しているのに仕事で使えない」という不安を、学習の順番と伴走体制で解消する構成です。

料金やサポート内容、よくある質問を後半で整理し、最後に無料セッションへつなげることで、検討中のユーザーが次の行動を取りやすい導線にしています。

工夫したポイント

  • 仕事英語に特化したコピーで、対象ユーザーを明確化
  • 90日ロードマップを軸に、サービス内容を視覚的に整理
  • 一般的な英会話との比較表で、英語コーチングの価値を説明
  • モデルケース・FAQ・料金表で、受講前の不安を解消
  • SP固定CTA、ハンバーガーメニュー、FAQ、デモモーダルをJavaScriptで実装

制作期間

13日
（構成設計・デザイン・コーディング・レスポンシブ対応含む）
※内容に応じて柔軟に対応可能です

使用技術

  • HTML / CSS（セマンティック設計）
  • JavaScript（メニュー・FAQ・スムーススクロール・デモモーダル・SP固定CTA）
  • レスポンシブ対応（スマホ / PC）
  • OGP・アクセシビリティ属性・画像最適化を意識した実装
